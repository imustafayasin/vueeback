## Vueeback, build with Node.js and MongoDB

---

##### Basic and usefull feedback app.

## Features

- Crud your feedbacks
- Comment and vote
- Advandec Filters
- Basic UI and understandable UX
- Generic error handling

## Usage

### Installation

```bash
npm install
```

### Start your local server

```bash
npm run start
```
